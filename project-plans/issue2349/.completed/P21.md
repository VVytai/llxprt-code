## Summary
Cross-package atomic migration of `getHistory` return type from Google-shaped `Content[]` to neutral `IContent[]`, deletion of Contract* payload types from `clientContract.ts`, and migration of 23+ CLI consumers + 5 core consumers.

## Changes
- **`packages/core/src/core/clientContract.ts`**: Deleted all `Contract*` types (ContractContent, ContractPart, ContractPartListUnion, ContractGenerateContentResponse, ContractSendMessageParameters, ContractGenerateContentConfig, ContractUsageMetadata). Introduced `AgentRequestInput` and `AgentClientMessageParams` as neutral replacements. `AgentClientContract.getHistory()` returns `Promise<IContent[]>`. `AgentClientContract.sendMessageStream` keeps positional args with `AgentRequestInput`. `AgentChatContract` methods use `AgentClientMessageParams`. Return types for `generateDirectMessage`/`generateContent` are `Promise<ModelOutput>`.
- **`packages/agents/src/core/client.ts`**: `AgentClient` methods migrated to neutral types — `addHistory(IContent)`, `getHistory(): Promise<IContent[]>`, `setHistory(IContent[])`, `startChat(IContent[])`, `generateContent(): Promise<ModelOutput>`, `generateDirectMessage(): Promise<ModelOutput>`.
- **`packages/agents/src/core/agenticLoop/loopHelpers.ts`**: `recordCancelledToolHistory` uses `iContentFromBlocks` directly (no `{role,parts}` construction).
- **`packages/agents/src/core/MessageStreamOrchestrator.ts`**: IDE context injection uses `iContentFromBlocks` directly.
- **`packages/agents/src/core/clientLlmUtilities.ts`**: `generateJson` and `generateContent` accept `IContent[]`.
- **`packages/cli/src/ui/commands/chatCommand.ts`**: CheckpointContent adapter for IContent→legacy format; UI history display reads `.blocks`.
- **`packages/a2a-server/src/agent/task.ts`**: `addHistory` passes neutral IContent directly.
- **Test helpers**: `realLoopHarness.ts`, `agenticLoop-test-helpers.ts` — types migrated from `Content[]` to `IContent[]`.
- **Test files**: `chatCommand.test.ts`, `copyCommand.test.ts`, `useAgentEventStream.loopIntegration.test.tsx` — expectations updated to IContent format.
